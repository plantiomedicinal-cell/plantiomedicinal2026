## 🌿 Sobre o Projeto

O **Raízes do Bem** é um projeto dedicado ao cultivo e à valorização de plantas medicinais, com foco em promover o uso consciente dos recursos naturais e resgatar conhecimentos tradicionais.

A proposta é conectar pessoas à natureza, incentivando práticas sustentáveis e oferecendo acesso a informações sobre o uso de ervas medicinais no dia a dia.
